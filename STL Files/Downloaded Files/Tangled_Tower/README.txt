# Summary

I redesigned the Tangled Tower for better printing experience and splitted it into printable parts. This is a quick modelling I did for my children. 

# Print Settings

Printer: Wanhao Maker 5s
Rafts: No
Supports: No
Resolution: 0,15
Infill: 20%
